## 📁 Project Structure

```plaintext
res://
├── Assets/       # Global assets (e.g., fonts, music, sound effects)
├── Entities/     # Game entity scenes and scripts
├── UI/           # In-game UI and menus
├── Utilities/    # Managers, global scripts, and event buses
├── Zones/        # Playable areas and scenes where gameplay entities are instanced
```
